OpenAI API Setup Guide for RadhaSphere Wallet

Note: This guide is for developers setting up the AI features in the RadhaSphere Wallet application.

Configuration Options

There are three ways to configure your OpenAI API key:

  1. Environment variable in .env file (Recommended for production)

    Create or edit the .env file in the root directory:

    OPENAI_API_KEY=your_actual_openai_api_key_here

    Then restart the development server with npm run dev.

  2. Config file (Recommended for development)

    Edit the app/config.ts file and replace the placeholder API key:

    export const config = {
  openaiApiKey: "your_actual_openai_api_key_here",
  aiModel: "gpt-3.5-turbo",
  maxTokens: 1000,
  temperature: 0.7
};

    Save the file and restart the development server.

  3. Direct code modification (Not recommended)

    As a last resort, you can directly modify app/api/ai/route.ts:

    const openai = new OpenAI({
  apiKey: "your_actual_openai_api_key_here",
});

    Warning: This method is not recommended as it risks exposing your API key if you push changes to version control.

Getting an OpenAI API Key

  1. Create an OpenAI account

    Visit https://platform.openai.com/signup and create an account if you don't already have one.

  2. Navigate to API keys

    Go to https://platform.openai.com/api-keys after logging in.

  3. Create a new secret key

    Click "Create new secret key", give it a name like "RadhaSphere Wallet", and copy the key.

Troubleshooting

If you're still having issues:

  • Make sure you've installed the OpenAI package with npm install openai
  • Verify your API key is correct and has sufficient credits
  • Check that you're using a supported model (gpt-3.5-turbo is recommended for most users)
  • Restart your development server after making any configuration changes
